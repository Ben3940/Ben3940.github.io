Landing Page

This is a clone of the chrome extension called Momentum. This web page will set the time (while continuously updating it) with respect to the user's system time. It will also store in local storage the user's name and their focus for that day, displaying the information provided back even if the browser is refreshed. Despite the small use of Bootstrap to make positioning of the text on mobile devices more appealing, vanilla HTML/CSS and JavaScript are the only things used in this chrome extension.

You can click on the image to see the extension in action.
